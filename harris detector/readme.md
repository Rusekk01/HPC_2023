# Harris detector algorithm

Реализована программная реализация алгоритма Харриса для детектирования углов на изображении. 

При одинаковых параметрах запуска точки, обнаруженные алгоритмами, совпадают.

## Время работы
| Разрешение    | CUDA, с             | CPU, с      |
|:-------------:|:---------------:                |:-------------:    |
| 574x600       | 0.257          | 3.36  |
| 1280x800       | 0.3917          | 12.99  |
| 2000x2828     | 0.923          | 125.52  |

<b>Вывод:</b> использование CUDA дает значительный прирост скорости даже при небольшом разрещении входного изображения.
